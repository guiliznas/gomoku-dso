Autores: Alfeu dos Santos, Guilherme de Liz e João Brognoli.

O projeto foi desenvolvido utilizando NetBeans 8.2, logo, recomenda-se
utilizar ele para executar o projeto.

!!! Atenção !!!
Para executar é necessário existir os arquivos config.txt e partidas.ser no
mesmo caminho que o programa está sendo executado.

Configurações atualizadas ao iniciar partida

========================
BUILD OUTPUT DESCRIPTION
========================

To run the project from the command line, go to the dist folder and
type the following:

java -jar "Gomoku.jar" 

To distribute this project, zip up the dist folder (including the lib folder)
and distribute the ZIP file.
